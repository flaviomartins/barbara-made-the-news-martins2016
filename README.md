# barbara-made-the-news-martins2016

```
news:                         Raw Google Search API responses
runs:                         QL-based baseline runs
topics:                       JSON formatted TREC topics with extra data
wikipedia_edits:              Special:Export XML Wikipedia page revisions
wikipedia_pageviews:          stats.grok.se daily Wikipedia page pageviews
list_of_wikipedia_bots.txt:   Names of bots filtered out from revision history
news.csv:                     News and extracted publishing dates in CSV
```
